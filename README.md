# Dannjs Cuda Addon

### Install

```
git clone --recurse-submodules https://github.com/matiasvlevi/dann-addon.git

cd dann-addon
npm i

cd Dann
git checkout Cuno
```

### Build

Build CUDA source with node-gyp (nvcc)

```
npm run build
```

### Run/Test

Run Javascript tests

```
npm run benchmark
npm run test
```
